# Report Template (каркас)

> Звіт = половина оцінки. Перевір приклад звіту в модулі Reporting і шаблон SysReptor.

## 1. Executive Summary
- Контекст і обсяг (scope)
- Загальна оцінка ризику
- Ключові висновки (нетехнічно, для керівництва)

## 2. Scope & Methodology
- In-scope активи, дати, обмеження
- Підхід (стадії пентесту)

## 3. Findings (на кожну вразливість)
- Назва + severity (CVSS)
- Опис
- Affected assets
- **Steps to Reproduce** (команди + скріншоти)
- Impact
- Remediation

## 4. Attack Chain Narrative
Послідовна історія: точка входу -> ескалація -> pivoting -> DA.

## 5. Appendices
- Список зібраних кредів
- Усі команди / артефакти

## Чеклист перед здачею
- [ ] Усі required-секції заповнені
- [ ] Кожен крок відтворюваний (команда + вивід)
- [ ] Скріншоти підписані й читабельні
